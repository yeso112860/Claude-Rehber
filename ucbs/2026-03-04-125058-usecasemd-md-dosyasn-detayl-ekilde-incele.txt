     e-İzin .NET Core 9.0 + Angular 18 - Claude Code Yapılandırma Planı

     Context

     e-İzin projesi (81 use case, 13 modül) için .NET Core 9.0 Clean
     Architecture + Angular 18 Signals + PrimeNG stack'inde Claude Code
     yapılandırması oluşturulacak. Amaç: tekrar eden kod üretim
     pattern'lerini skill'lerle otomatikleştirmek, kod kalitesini
     agent'larla denetlemek, kurallarla tutarlılığı sağlamak.

     Teknoloji kararları: Clean Architecture (CQRS + MediatR), Angular
     Signals + PrimeNG, PostgreSQL + EF Core, JWT auth, katman bazlı
     solution, şimdilik test yok.

     Oluşturulacak Dosyalar
